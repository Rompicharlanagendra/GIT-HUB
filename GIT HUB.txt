Commands for GIT to perform:-

1)git init
2)git add filename or .
3)git status
4)git commit -m "message"
5)git log
6)git remote add origin "<URL to repository>"    --- Local repository to remote repository

7)git branch  -----> To check the current working branch and no of branches.
8)git branch branchname  ----> To create the branch 
9)git checkout branch
